# **App Name**: Rasa Kitchen

## Core Features:

- Cinematic Hero Section: Full-screen hero displaying animated WebP background, brand name 'RASA KITCHEN' in elegant typography, tagline, and an 'Explore Our Menu' CTA button.
- Responsive Navigation Bar: Top navbar with 'RASA KITCHEN' branding and links ('Menu', 'About', 'Recipe', 'Nutrition', 'Reviews', 'FAQ', 'Contact') for smooth scrolling to relevant sections.
- Dynamic Content Display: Dedicated sections for 'About Us' story, 'Our Menu' with meal variant cards, 'Ingredients' grid, 'Step-by-step Recipe', 'Nutrition Facts' label, 'Reviews' testimonials, and 'FAQ' accordion.
- AI Image Generation Tool: Utilize a generative AI tool to create stunning 3D rendered images for specific menu items like Saffron Rice and Aloo Sabzi, based on textual descriptions, maintaining the high-end visual theme.
- Interactive Elements & Call-to-Actions: Functional CTA buttons (e.g., 'Explore Our Menu', 'Book a Table Tonight'), and an interactive FAQ accordion for an engaging user experience.
- Persistent Footer: A visually consistent black footer containing the brand logo/name, essential links (About, Contact, Privacy, Terms), social media icons, and copyright information.

## Style Guidelines:

- Primary Color: A vibrant Saffron Gold (#FF9933) to provide warmth and richness, representing authentic flavors and premium quality, especially prominent for CTAs and interactive elements. HSL(30, 100%, 60%).
- Background Color: Pure Black (#000000) for a luxurious, cinematic dark mode experience as specified.
- Accent Color: A deep, muted red (#CC1A1A) for subtle highlights, small graphic elements, or secondary text, complementing the saffron gold. HSL(0, 70%, 40%).
- Headlines will use 'Playfair' (serif, geometric) for an elegant, high-end, and cinematic feel. Body text will use 'PT Sans' (sans-serif, humanist) for clean, highly readable content suitable for longer paragraphs like the 'About Us' section and recipes.
- Clean, simple emoji icons will be used for ingredient lists. Professional social media icons will be integrated into the footer. Focus on minimalist and high-quality iconography to maintain the premium aesthetic.
- Single-page, scroll-based layout with a full-screen hero. Content organized into distinct, visually separated sections using cards for meal variants, ingredients, and reviews, an accordion for FAQ, and subtle gold dividers between major sections to enhance visual flow on a pure black background.
- The site will feature large animated WebP visuals in the hero section and for specific menu items. Smooth scrolling will be implemented between sections. Subtle hover animations and transitions on interactive elements like buttons and menu cards will add to the cinematic experience.
