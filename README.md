# Rasa Kitchen

Authentic Indian Flavors, Cinematic Experience

A premium single-page restaurant website for Rasa Kitchen — 
an authentic Indian meal brand that brings restaurant-quality 
dishes to your home with a modern, cinematic presentation.

---

## Features

- Cinematic hero section with animated karahi WebP video
- Curated menu with 3 signature dishes
- Brand story and culinary heritage section
- Ingredient showcase with visual cards
- Step-by-step recipe with cooking timings
- Nutrition facts card
- Customer reviews and testimonials
- FAQ accordion section
- Fully responsive on all devices
- Dark mode design with saffron gold accents

---

## Tech Stack

- Next.js 14
- Tailwind CSS
- TypeScript
- Animated WebP via Google Flow and Whisk
- Deployed on Vercel
- Media hosted on GitHub

---

## Design

Background: #000000 Pure Black

## Getting Started

```bash
git clone https://github.com/RIYAKUMARI001/Meal.git
cd Meal
npm install
npm run dev
```

## AI Tools Used

- Google Whisk — first and last frame generation
- Google Flow Veo 3.1 — karahi animation video
- Ezgif — MP4 to WebP conversion
- Firebase Studio — website code generation
- Google Gemini — dish photography images
